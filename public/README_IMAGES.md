# 社交媒体图片说明

## 需要创建的图片

### 1. OG Image (og-image.jpg)
- 尺寸: 1200x630 像素
- 内容建议:
  - Logo和网站名称 "QuoteGenerator"
  - 标语: "Transform Bible Quotes into Beautiful AI Art"
  - 简洁的背景，可以使用圣经相关的设计元素
  - 网站URL: quotegenerator.com

### 2. Twitter Card Image (twitter-image.jpg)
- 尺寸: 1200x600 像素
- 内容建议:
  - 类似OG图片但针对Twitter优化
  - 更突出的视觉元素
  - 可以包含一个示例的圣经金句

### 3. 默认分享图示例 (default-share-bg.png)
- 尺寸: 1080x1080 像素 (正方形，适合Instagram)
- 内容建议:
  - 优雅的背景设计
  - 留有文字空间用于放置圣经金句
  - 可以创建多个不同风格的背景供用户选择

## 创建工具建议
- 使用 Canva、Adobe Express 或 Figma
- 或使用AI工具如 Midjourney、DALL-E 生成
- 确保图片质量高且加载速度快
